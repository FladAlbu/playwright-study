## Sobre
Repositório do treinamento
## Stacks
- Playwright
- TypeScript
- Tesults

## Rodando
1. Clonar o repositório, instalar as dependências
```
npm install
```

2. Executar testes em Headless
```
npx playwright test 
```

3. Executar ver o relatório dos testes
```
npx playwright show-report
```
